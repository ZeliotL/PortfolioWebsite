Medtech Bioprinter X

A visual showcase of a Bioprinter and its uses

Tools used:

Blender, Visual studio, webbrowser

Project type & Timeframe:

School Project (group), 4 Weeks

What does this showcase?

I can work in a group using blender and collaborating on different blender systems. I can work with the various rigging and animation techniques possible inside of blender, this includes bone constraints, animation drivers, armature creation, weight painting and geometry attributes. Being able to use geometry nodes systems and implement it with a 3D model armature/rig.
I can create websites from scratch with custom functionality using languages like HTML, CSS and Javascript. Then publish them onto the internet for anyone to view and interact with.
