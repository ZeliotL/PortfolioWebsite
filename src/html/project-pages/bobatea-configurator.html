Bobatea Configurator

configure bobatea with various flavours and styles

Tools used:

Unity, Blender, Substance Designer, Material Maker

Project type & Timeframe:

School Project (alone), 4 Weeks

What does this showcase?

I can work inside Unreal engine and Unity in a solo environment, creating systems via Blueprints, connect them to custom made shaders and alter parts of a model and its shaders during runtime. I can also set up cameras and cinematics to showcase objects at their best. Create optimized custom assets from the ground up designed for a 3D environment.


More in depth


The Unity version

(Unreal Engine version further down)

I wanted to make both an Unreal and Unity version of this shader. Below you can see the Unity version, it has some more advanced pearl movement and a completely shader based straw!

The Unreal Engine 5 version

The Unreal Engine 5 version was made before the Unity version. This version has a realtime menu you can use to adjust the cups content and a cinemachine system that allows for cinematic shots and b-rolls of the boba drink.

Configurator preview

full preview of what and how you can customize the drink.
You can spin around the cup to view it from any angle, which also moves the pearls around. The menu is also interactive and when you click on a option the drink changes during runtime.

Bobatea Shader Graph

The boba shader uses a 3D perlin noise that moves around depending on how you move the cup. You can also change various variables like the height of the pearls or their overall size, there's also options to swap out textures for the sauce drizzles.

Boba Actor

The actor for the bobatea cup. It has mouse interactivity so you can spin it around and it gradually slows down. It also has a dynamic runtime material which is what allows it to change colors based on what you click on the menu.

Configurator Menu Editor

The boba menu has an editor of its own where you can add more buttons and very easily swap out some parameters, like name and color, to make it change the corresponding part of the boba drink.

Dynamically adjustable straw visuals

Unlike the Unity version of this project, the Unreal Engine 5 version has a blender material for the straw that is dynamically adjustable and then baked out into a texture. This saves a little bit on performance but may look worse and isnt as adjustable afterwards.

Core functionality blueprints

Blueprints of some important core functionality like setting up dynamic materials for the boba cup, connecting the menu options to that dynamic material, the functionality for the mouse to be able to click and interact with the menu and drink.
