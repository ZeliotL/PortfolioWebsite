Sticker Training Program

A Unity application training users how to use sticker tools in VR

Tools used:

Unity, Blender, Substance Designer

Project type & Timeframe:

School Project (alone), 4 Weeks

In this project I focused on creating a interactive application using Unity with C#. That included things like interactivity between the user, items and tools, advanced visuals that clearly convey whats going on, such as the printer printing stickers onto the sheets, the cutter machine cutting out each sticker, the scanner projecting scan lines and scans barcodes.
One important aspect when creating the individual systems was to make sure they can communicate with each other as to prevent weird buggy behaviour. To achieve this I created a system manager and added different states to the various tools and objects.
I also tried to figure out how I can guide the user with visuals and environment cues only as a means to avoid using big chunks of text everywhere to explain how things worked. This is why theres pieces of paper on the ground next to the paper container, or a pre-packaged cardboard box next to the individual components.
Most 3D models are from online sources, then altered and refined for this project. Everything else in this project is created entirely by me.
