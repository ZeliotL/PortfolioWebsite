Sticker Training Program

A Unity application training users how to use sticker tools in VR

Tools used:

Unity, Blender, Substance Designer, Substance Painter, Photoshop

Project type & Timeframe:

School Project (alone), 4 Weeks

What does this showcase?

I can use Unity to create full system loops and workflows. Visualize whats happening using effects, lighting techniques and custom shaders. Integrate them with the implemented systems and create a complete product. VR and environment optimization to make sure it runs smoothly.
Work with premade assets then optimize and alter them for the needs of the project. Create textures using substance designer, substance painter and blender shaders.


More in depth

In this project I focused on creating a interactive application using Unity with C#. That included things like interactivity between the user, items and tools, advanced visuals that clearly convey whats going on, such as the printer printing stickers onto the sheets, the cutter machine cutting out each sticker, the scanner projecting scan lines and scans barcodes.
One important aspect when creating the individual systems was to make sure they can communicate with each other as to prevent weird buggy behaviour. To achieve this I created a system manager and added different states to the various tools and objects.
I also tried to figure out how I can guide the user with visuals and environment cues only as a means to avoid using big chunks of text everywhere to explain how things worked. This is why theres pieces of paper on the ground next to the paper container, or a pre-packaged cardboard box next to the individual components.
Most 3D models are from online sources, then altered and refined for this project. Everything else in this project is created entirely by me.

Sticker cutter zoomed in

You can see that the cutting machine actually cuts out the stickers from the sheet, and then you can take them out.
It also adapts and cuts the stickers out correctly no matter the position and rotation of the sheet.

Cutting Shader

A shader is being used for the visuals of the stickers being cut out of the sheet.
In the first video you can see how depending on which value is being set, it gradually progresses how many of the stickers have been cut. There is also a value that goes from 0 to 1 to show how much each individual sticker has been cut out.
Both of these values are changed during runtime via a C# script. The sticker cutter controls these values and moves around the sticker at the same time, giving the sense of the sticker actually being cut out.

Sticker Sheet States

The sticker sheet uses alpha cutout to remove the sticker while being cut out by the machine. To avoid the sticker from vanishing the cutting machine spawns in a individual sticker which is then able to get take off from the sheet.

The cutter shader also needs to orient itself to the cutting machine so when it moves around each sticker, the sheet also removes the gradual cutout in the right position.

Sticker Cutting Machine Overview

The sticker cutter is a huge part of this project and works alongside many of the other components to cut out stickers from sheets and make them separate so the user can pick them out of the sheet afterwards.

Cutter Display

The display shows relevant information to the user for what state the machine is in.
These states are mostly pretty obvious but "wrong placement" exists because the cutting machine checks if the paper sheet is properly within its cutting area or not.

Cutter Lamp

The red lamp on the cutting machine idicates to the user that the machine is active when turned on. To give it a little bit more visual flare i made a shader for it with psuedo randomness to make it flicker every now and then.

Delivery Box

The delivery box has two states: closed and open. When its open you can put items inside of it, when you close it, the contents and the open box mesh gets hidden and the closed box is turned on. When you open it again, the process is reversed and the box can be filled with more stickers.

Printer

The printer changes the state of the sticker sheet to add the stickers. The sheet is never actually modified and its only a simple variable change in its shader. To give the illusion of it being printed out, the sheet sticks out from the machine on the backside, this is never visible for the user so it works perfectly without unecessary functionality to keep it within the printer at all times.

Scanner

The scanner can scan barcodes, they appear on the sticker sheets to indicate which order it is, and also on the delivery receipts for easy handling of which order it belongs to.

Blender Scene

This is the blender scene where i kept all of the models for the project. Here i modifies them for my needs by altering the mesh itself, adding or changing textures, optimizing and removing unecessary parts to keep everything running smoothly.

Cutting Machine Textures

I Created a dynamic metallic material for the metallic parts of the cutting machine. It's then baked out as a texture to be used within Unity. I also used Substance Designer and Painter to create the green cutting pad for the machine. Substance Designer was used for the main grid, then in Substance Painter i created the other measurement lines as needed.
